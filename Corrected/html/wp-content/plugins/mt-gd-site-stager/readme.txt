Easily create a staging site for your wordpress website.

== Description ==

This plug-in lets you easily create a staging site where you can make and test
changes, and then sync them into your live site.

Key features:

* easy to use
* it works!

== Installation ==

1. Sign up for Premium Wordpress Service at http://mediatemple.net and the
plugin is installed for you.
2. Make a staging site!
